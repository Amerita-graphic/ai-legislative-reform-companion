Try asking:
  • "Summarise the enforcement gaps in the Fiji Tobacco Control Act"
  • "Which clauses relate to licensing in the Kava Amendment Bill?"
  • "What penalties exist in the Nauru Liquor Control Act?"

3. About This Project

This prototype demonstrates how AI can support legislative reform in Pacific health systems by:

  • Generating legislative briefs
  • Creating evidence → clause → amendment cross‑walks
  • Producing visual explainers for senior audiences
  • Providing a chatbot for rapid clause analysis
  • Automating workflows using Relay.app
